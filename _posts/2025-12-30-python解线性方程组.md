## <center>代码如下</center>

　　用于求解三角晶格的J

```python

import numpy as np

# 系数矩阵 A
A = np.array([
    [18, 18, 18, 1],
    [-6, -6, 18, 1],
    [-6, 2, 2, 1],
    [-2, -6, 2, 1]
])

# 从用户输入右端向量 b
print("请输入右端常数向量 b 的4个数（以空格分隔，例如：1 2 3 4）:")
b = np.array(list(map(float, input().split())))

# 检查维度是否正确
if len(b) != 4:
    raise ValueError("必须输入4个数！")

# 求解线性方程组 Ax = b
x = np.linalg.solve(A, b)

# 输出结果
print("\n方程组的解为：")
for i, val in enumerate(x, start=1):
    print(f"x{i} = {val:.6f}")


```
