## <center>步骤</center>

(1) 完成一次拟合良好的wannier，注意INCAR中加入 LWRITE_UNK = .TRUE. 输出Unk文件

```shell

LWRITE_UNK = .TRUE.

```


(2) 产生nnkp文件，输入


```shell

wannier90.x -pp wannier90.1

```

```shell

wannier90.x -pp wannier90.2

```


(3) 在win文件中加入


```shell

wannier_plot = .true.
wannier_plot_supercell=3
wannier_plot_format = cube
wannier_plot_mode = crystal
wannier_plot_radius = 2.5
wannier_plot_scale = 1.0

```


(4) 运行wannier90输出cube


```shell

wannier90.x wannier90.1

```
```shell

wannier90.x wannier90.2

```


(5) 使用vesta打开产生的cube文件
