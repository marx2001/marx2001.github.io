## <center>参数注释</center>

计算表面态

```shell

&TB_FILE
Hrfile = 'wannier90_hr.dat'   ! 指定Wannier90输出的哈密顿矩阵文件
Package = 'VASP'              ! 说明原始计算来自VASP
/

&CONTROL
BulkBand_calc         = F      ! 是否计算体能带 (F=否, T=是)
SlabSS_calc           = T      ! 是否计算表面态 (T=是)
Z2_3D_calc            = F      ! 是否计算三维体系的Z2拓扑不变量 (F=否)
/ 

&SYSTEM
SOC = 1                        ! 是否包含自旋轨道耦合 (SOC=1 表示开启)
E_FERMI = -1.1651              ! 费米能级 (单位 eV)
Numoccupied = 6                ! 占据能带的数目
/

&PARAMETERS
Eta_Arc = 0.001                ! 极小值，用作展宽因子 (类似能量展宽) 
E_arc = -0.05                  ! 计算费米弧时对应的能量 (单位 eV)
OmegaMin = -1.3                ! 能量范围下限 (绘制能带图时的最小能量)
OmegaMax =  1.3                ! 能量范围上限 (绘制能带图时的最大能量)
OmegaNum = 401                 ! 能量采样点数量 (能量步长的分辨率)
Nk1 = 101                      ! k点数 (方向1)，建议使用奇数以包含Γ点
Nk2 = 101                      ! k点数 (方向2)，建议使用奇数以包含Γ点
Np  = 2                        ! k点数 (方向3或平面数)，建议使用奇数
/

SURFACE                        ! 定义表面切面（此处为zigzag方向）
 1  0  0                       ! 表面法向的第一行 (对应晶格矢量 a1)
 0  0  1                       ! 第二行 (对应晶格矢量 a2)
 0  1  0                       ! 第三行 (定义表面坐标系)
 
KPATH_BULK                     ! 定义体能带计算的k点路径
3                              ! k线的数量 (仅在计算体能带时使用)
G 0.00000  0.00000 0.0000  Y 0.50000  0.00000 0.0000
Y 0.50000  0.00000 0.0000  M 0.33333  0.33333 0.0000
M 0.33333  0.33333 0.0000  G 0.00000  0.00000 0.0000

KPATH_SLAB                     ! 定义二维体系表面态的k点路径
2                              ! k线数量（用于二维表面态）
X  0.5  0.0  G  0.0  0.0       ! 第一条k线路径 (X→Γ)
G  0.0  0.0  X  0.5  0.0       ! 第二条k线路径 (Γ→X)

LATTICE                        ! 晶格参数定义
Angstrom                       ! 单位为 Å
     2.1017795    -3.6403930     0.0000000
     2.1017795     3.6403930     0.0000000
     0.0000000     0.0000000    15.0000000  ! z方向真空层厚度

ATOM_POSITIONS
2                               ! 原子数目
Cartisen                        ! 原子坐标类型（Direct为分数坐标，Cartisen为笛卡尔坐标）
Bi       2.1017795     1.2134643    10.7681236
Bi       2.1017795    -1.2134643     9.0367958

PROJECTORS
3 3                             ! 定义投影子数量（每个原子3个投影轨道）
Bi pz px py                     ! 第一个Bi的投影轨道
Bi pz px py                     ! 第二个Bi的投影轨道

!WANNIER_CENTRES                ! （可选项）从wannier90.wout复制Wannier中心位置
!Cartesian                      ! 表示使用笛卡尔坐标系


```
