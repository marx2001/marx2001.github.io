## <center>说明</center>

本文整理用于**块体材料 Dzyaloshinskii--Moriya interaction（DMI）第一性原理计算与提取**的代表性文献。重点不是介绍 DMI 的物理起源，而是给后续实际计算提供文献依据，包括：

```shell
1. 四态能量映射法 four-state energy mapping；
2. spin spiral / generalized Bloch 方法；
3. Green's function / magnetic force theorem 方法；
4. Wannier / TB2J 自动提取 J、D、各向异性交换的方法；
5. 已经在块体材料中使用过的典型案例，例如 BiFeO3、Cu2OSeO3、FeGe、Mn1-xFexGe。
```

对于块体材料，DMI 一般不应只写成一个标量 \(D\)，而应写成每一类磁性键的 DMI 矢量：

\[
H_\mathrm{DMI}
=
\sum_{ij}\mathbf{D}_{ij}\cdot
\left(\mathbf{e}_i\times\mathbf{e}_j\right)
\]

或者根据文献约定写成带负号的形式：

\[
H_\mathrm{DMI}
=
-\sum_{ij}\mathbf{D}_{ij}\cdot
\left(\mathbf{e}_i\times\mathbf{e}_j\right)
\]

因此，在引用文献或写计算方法时，必须明确自己的哈密顿量符号约定。

## <center>方法分类</center>

### 1. 四态能量映射法

四态能量映射法适合从总能差中提取某一对磁性原子的交换矩阵。对于两个自旋 \(i,j\)，一般写成：

\[
H_{ij}
=
\mathbf{S}_i
\mathbf{J}_{ij}
\mathbf{S}_j
\]

其中 \(\mathbf{J}_{ij}\) 是 \(3\times3\) exchange matrix。其反对称部分对应 DMI：

\[
D^x_{ij}
=
\frac{J^{yz}_{ij}-J^{zy}_{ij}}{2}
\]

\[
D^y_{ij}
=
\frac{J^{zx}_{ij}-J^{xz}_{ij}}{2}
\]

\[
D^z_{ij}
=
\frac{J^{xy}_{ij}-J^{yx}_{ij}}{2}
\]

这个方法的优点是公式直观，适合写在支撑材料中。缺点是对于复杂块体材料，超胞里往往含有很多等价或不等价的磁性键，容易把不同路径的 DMI 混在一起。因此使用时必须明确：

1. 选择哪一对磁性原子；
2. 超胞中这一类等价键的数量 N；
3. 是否存在多个不等价 V--V、Fe--Fe、Cu--Cu 或其他磁性键；
4. 哈密顿量中 DMI 项的正负号约定；
5. 最终提取的是单键 \(\mathbf{D}_{ij}\)，还是等效总 DMI。


### 2. Spin spiral 方法

spin spiral 方法适合提取连续模型中的有效 DMI。对于小 \(q\) 附近的自旋螺旋，可以写成：

\[
E(q)
=
Aq^2
\pm Dq
\]

或者用左右手螺旋能量差：

\[
\Delta E(q)
=
E_\mathrm{CW}(q)-E_\mathrm{CCW}(q)
\]

在小 \(q\) 极限下，\(\Delta E(q)\) 与 \(q\) 的线性项给出有效 DMI。这个方法适合：

1. 块体 B20 手性磁体；
2. BiFeO3 这类长周期 spin cycloid；
3. 想得到连续模型参数 A 和 D，而不是每条键的 \(\mathbf{D}_{ij}\)；
4. 后续要和微磁模拟或长周期螺旋周期联系。


### 3. Green's function / Wannier / TB2J 方法

Green's function 方法基于 magnetic force theorem 和局域刚性自旋旋转微扰。它的优势是可以从 Wannier Hamiltonian 或局域轨道 Hamiltonian 中自动提取远距离磁相互作用，包括：

\[
J_{ij},\quad
\mathbf{D}_{ij},\quad
\Gamma_{ij}
\]

对于复杂块体体系，这条路线通常比手动四态法更稳妥。尤其是当磁性原子很多、交换路径复杂、近邻壳层很多时，推荐优先考虑 TB2J 或同类 Green's function 程序。

### 1. First-principles calculations for Dzyaloshinskii--Moriya interaction

```text
Hongxin Yang, Jinghua Liang, and Qirui Cui,
First-principles calculations for Dzyaloshinskii--Moriya interaction,
Nature Reviews Physics 5, 43--61 (2023).
DOI: 10.1038/s42254-022-00529-0
```

这是一篇 DMI 第一性原理计算综述，适合放在方法部分开头作为总述文献。文章系统讨论了 DMI 的第一性原理计算路线，包括：

```shell
1. supercell energy mapping；
2. generalized Bloch theorem / spin spiral；
3. Green's function；
4. Berry phase / spin current 相关表述；
5. 不同材料体系中的 DMI 诱导手性磁结构。
```

建议用途：

```shell
如果论文中需要一句话说明“DMI 可以通过多种第一性原理方法计算”，优先引用这篇。
```

### 2. Magnetic properties and energy-mapping analysis

```text
Hongjun Xiang, Changhoon Lee, Hyo-Jun Koo, Xiangdong Gong, and Myung-Hwan Whangbo,
Magnetic properties and energy-mapping analysis,
Dalton Transactions 42, 823--853 (2013).
DOI: 10.1039/C2DT31662E
```

这篇是能量映射法的经典综述，主要讨论如何从第一性原理总能中映射得到磁性哈密顿量参数。虽然它不只讲 DMI，但对于解释 exchange、single-ion anisotropy、DMI 等磁参数的能量映射思想非常有用。

建议用途：

```shell
1. 说明为什么可以用 DFT 总能差提取磁参数；
2. 说明 spin Hamiltonian 的建立过程；
3. 给 J、D、K 等磁参数拟合提供方法依据。
```

### 3. Generic four-state energy mapping onto a Heisenberg spin Hamiltonian

```text
Denis Šabani, Cihan Bacaksız, and M. V. Milošević,
Ab initio methodology for magnetic exchange parameters:
Generic four-state energy mapping onto a Heisenberg spin Hamiltonian,
Physical Review B 102, 014457 (2020).
DOI: 10.1103/PhysRevB.102.014457
```

这篇文章系统推导了通用四态能量映射公式。它的核心价值是：不是只提取一个标量 \(J\)，而是提取完整的 exchange matrix。DMI 对应 exchange matrix 的反对称部分。

建议用途：

1. 如果要在支撑材料中写四态映射公式，优先引用这篇；
2. 如果要提取 \(\mathbf{D}_{ij}=(D^x,D^y,D^z)\)，这篇比普通 J 映射文献更合适；
3. 对复杂低对称体系、块体材料、二维材料都可作为方法依据。

3. 想把第一性原理 DMI 与 Monte Carlo / spin dynamics 联系。

### 9. TB2J: A python package for computing magnetic interaction parameters

```text
Xu He, Nicole Helbig, Matthieu J. Verstraete, and Eric Bousquet,
TB2J: A python package for computing magnetic interaction parameters,
Computer Physics Communications 264, 107938 (2021).
DOI: 10.1016/j.cpc.2021.107938
```

TB2J 是一个自动计算磁相互作用参数的 Python 程序，可以从 Wannier90 或局域轨道 Hamiltonian 中提取：

1. isotropic exchange \(J_{ij}\)；
2. anisotropic exchange；
3. Dzyaloshinskii--Moriya interaction \(\mathbf{D}_{ij}\)；
4. 可用于后续 magnon、spin dynamics、Monte Carlo 等计算。


这篇对于复杂块体材料非常实用。因为块体材料中磁性原子多、交换路径多，手动四态能量映射很容易遗漏或混合不同键的贡献。TB2J 可以更系统地输出不同距离、不同原子对之间的磁相互作用参数。

建议用途：

```shell
如果你的体系结构复杂，优先考虑 Wannier90 + TB2J，而不是手动构造大量非共线四态。
```

## <center>针对块体材料的推荐引用组合</center>

如果目标是写“块体材料 DMI 的计算方法”，推荐按下面方式组合引用。

### 情况一：使用四态能量映射法

```text
Xiang et al., Dalton Trans. 42, 823--853 (2013).
Šabani et al., Phys. Rev. B 102, 014457 (2020).
Xu et al., Phys. Rev. B 99, 104420 (2019).
```

推荐写法：

```text
The magnetic exchange parameters, including the isotropic exchange and Dzyaloshinskii--Moriya interaction, can be extracted by mapping first-principles total energies onto an atomistic spin Hamiltonian. In particular, the DMI vector corresponds to the antisymmetric part of the exchange matrix obtained from four-state energy mapping.
```

### 情况二：使用 spin spiral 方法

```text
Xu et al., Phys. Rev. B 103, 214423 (2021).
Loh and Gan, AIP Adv. 7, 056412 (2017).
```

推荐写法：

For long-wavelength spiral states, the effective DMI can be evaluated from the linear-in-\(q\) contribution to the spin-spiral energy dispersion.


### 情况三：使用 Wannier / TB2J / Green's function 方法

```text
He et al., Comput. Phys. Commun. 264, 107938 (2021).
Mahfouzi and Kioussis, Phys. Rev. B 103, 094410 (2021).
Koretsune et al., Sci. Rep. 5, 13302 (2015).
```

推荐写法：

```text
The pairwise exchange parameters were further evaluated using a Green's-function-based magnetic force theorem approach, which allows the calculation of isotropic exchange, anisotropic exchange, and Dzyaloshinskii--Moriya interaction from Wannier Hamiltonians.
```

## <center>实际计算时需要注意的问题</center>

### 1. 先判断 DMI 是否由对称性允许

DMI 存在的必要条件不是“块体”或“二维”，而是：

1. 体系存在 SOC；
2. 对应磁性原子对之间的交换路径没有反演中心；
3. 该键的局域对称性允许对应方向的 \(\mathbf{D}_{ij}\) 分量。


即使整个晶体是非中心对称，也不代表所有磁性键都有 DMI。反过来，即使整体晶体具有反演对称，也可能在局域非中心对称子晶格中讨论 hidden / staggered DMI，但这时需要更谨慎地处理对称性和磁结构。

### 2. 块体材料不能只看层内最近邻

块体材料至少要区分：

```shell
1. 层内最近邻；
2. 层内次近邻；
3. 层间近邻；
4. 不等价磁性原子之间的交换路径；
5. 不同晶向上的 DMI 矢量方向。
```

### 3. 四态法要除以等价键数量

如果四态能量差对应的是一个超胞总能，则提取单键 DMI 时必须除以该超胞中对应等价键数量 \(N\)。否则得到的是总贡献，不是单键 \(\mathbf{D}_{ij}\)。

### 4. 哈密顿量符号要统一

常见写法有两种：

\[
H_\mathrm{DMI}
=
\sum_{ij}\mathbf{D}_{ij}\cdot
(\mathbf{e}_i\times\mathbf{e}_j)
\]

和

\[
H_\mathrm{DMI}
=
-\sum_{ij}\mathbf{D}_{ij}\cdot
(\mathbf{e}_i\times\mathbf{e}_j)
\]

两种写法会导致 \(D\) 的符号相反。因此，最终论文中必须明确采用哪一种定义。

## <center>对复杂块体材料的建议</center>

对于类似 Cs3V9Te13 这种 V 原子较多、交换路径复杂、层状但仍属于块体周期结构的体系，更推荐优先采用：

```shell
Wannier90 + TB2J
```

原因是：

1. 可以一次性输出多个近邻壳层的 \(J_{ij}\) 和 \(\mathbf{D}_{ij}\)；
2. 能区分不同 V--V 键、不等价 V 原子以及层内/层间路径；
3. 比手动四态映射更不容易混入多个键的贡献；
4. 输出结果更适合后续导入 Spirit、Vampire 或 Monte Carlo / spin dynamics 程序。


如果只是为了在支撑材料中给出一个清楚、可复现的 DMI 提取公式，则可以采用：

```shell
四态能量映射法 + Šabani 2020 + Xu 2019 BiFeO3
```

如果目的是得到连续模型中的有效 DMI 常数，并与螺旋周期或 skyrmion 尺寸联系，则可以采用：

```shell
spin spiral 方法 + Xu 2021 BiFeO3 + Loh 2017 FeGe
```
