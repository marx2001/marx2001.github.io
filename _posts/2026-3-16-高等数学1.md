## 高等数学第一章《函数与极限》第一节：映射与函数 定义与要点整理

## 一、总框架

1. **章节主题**  
   第一章的主题是“函数与极限”。

2. **三个核心关键词**  
   1. 函数——研究对象  
   2. 极限——研究方法  
   3. 连续——研究桥梁

3. **第一节内容**  
   第一节主要包括三部分：  
   1. 集合  
   2. 映射  
   3. 函数

## 二、集合

### 1. 集合的定义

1. **集合**  
   具有某种特定性质的事物的总体称为集合。

2. **元素**  
   组成集合的事物称为元素。

3. **空集**  
   不含任何元素的集合称为空集，记作 `∅`。

### 2. 元素与集合的关系

1. **属于**  
   若元素 `a` 属于集合 `M`，记作：  
   `a ∈ M`

2. **不属于**  
   若元素 `a` 不属于集合 `M`，记作：  
   `a ∉ M`

### 3. 常见集合记号

1. **M\***  
   表示数集 `M` 中去掉 `0` 后的集合。

2. **M+**  
   表示数集 `M` 中去掉 `0` 和负数后的集合。

### 4. 集合的表示法

1. **列举法**  
   按某种方式把集合中的全体元素一一列出。

2. **描述法**  
   用元素所具有的共同特征来表示集合。

### 5. 常见数集

1. 自然数集：`N`  
2. 整数集：`Z`  
3. 有理数集：`Q`  
4. 实数集：`R`

### 6. 区间

1. **开区间**  
   `(a, b) = {x | a < x < b}`

2. **闭区间**  
   `[a, b] = {x | a ≤ x ≤ b}`

3. **半开区间**  
   `[a, b)`  
   `(a, b]`

4. **无限区间**  
   `[a, +∞)`  
   `(-∞, b]`  
   `(-∞, +∞) = R`

### 7. 邻域

1. **点 a 的 δ 邻域**  
   `U(a, δ) = {x | |x - a| < δ}`

2. **点 a 的去心 δ 邻域**  
   `U°(a, δ) = {x | 0 < |x - a| < δ}`

3. **左 δ 邻域**  
   `(a - δ, a)`

4. **右 δ 邻域**  
   `(a, a + δ)`

5. **邻域中的两个基本概念**  
   1. `a` 称为邻域中心  
   2. `δ` 称为邻域半径

### 8. 集合之间的关系

1. **子集**  
   若集合 `A` 的任一元素都属于集合 `B`，则称 `A` 是 `B` 的子集，记作：  
   `A ⊂ B`

2. **集合相等**  
   若 `A ⊂ B` 且 `B ⊂ A`，则称 `A` 与 `B` 相等，记作：  
   `A = B`

3. **基本结论**  
   1. `A ⊂ A`  
   2. `A = A`  
   3. `∅ ⊂ A`  
   4. 若 `A ⊂ B`，且 `B ⊂ C`，则 `A ⊂ C`

### 9. 集合的运算

1. **并集**  
   `A ∪ B = {x | x ∈ A 或 x ∈ B}`

2. **交集**  
   `A ∩ B = {x | x ∈ A 且 x ∈ B}`

3. **差集**  
   `A \ B = {x | x ∈ A 且 x ∉ B}`

4. **余集（补集）**  
   若 `B ⊂ A`，则 `B` 在 `A` 中的余集记作：  
   `B_A^c = A \ B`

5. **直积**  
   `A × B = {(x, y) | x ∈ A，y ∈ B}`

6. **特例**  
   `R × R = R^2`  
   表示平面上的全体点集。

### 10. 集合部分要点总结

1. **必须掌握的基础内容**  
   1. 元素与集合的关系符号  
   2. 空集  
   3. 区间表示  
   4. 邻域、去心邻域、左邻域、右邻域  
   5. 子集  
   6. 并集、交集、差集、补集、直积

2. **学习作用**  
   集合是高等数学中最基础的语言工具，后面讨论函数、极限、连续时都会反复用到。

## 三、映射

### 11. 映射的定义

1. **映射**  
   设 `X`、`Y` 是两个非空集合，若存在一个对应规则 `f`，使得对每个 `x ∈ X`，都有唯一确定的 `y ∈ Y` 与之对应，则称 `f` 为从 `X` 到 `Y` 的映射，记作：  
   `f: X → Y`

### 12. 像与原像

1. **像**  
   若 `y = f(x)`，则 `y` 称为元素 `x` 在映射 `f` 下的像。

2. **原像**  
   元素 `x` 称为元素 `y` 的原像。

### 13. 定义域与值域

1. **定义域**  
   集合 `X` 称为映射 `f` 的定义域。

2. **值域**  
   `f(X) = {f(x) | x ∈ X}`  
   称为映射 `f` 的值域，它是 `Y` 的一个子集。

### 14. 映射的三要素

1. **映射的三要素包括**  
   1. 定义域  
   2. 对应规则  
   3. 值域

### 15. 映射的基本性质

1. **唯一性**  
   对每一个 `x`，它的像 `y` 必须唯一。

2. **原像不唯一**  
   某个 `y` 的原像不一定唯一。

### 16. 单射、满射、双射

1. **满射**  
   若 `f(X) = Y`，则称 `f` 为满射。

2. **单射**  
   若 `x1 ≠ x2` 时，总有 `f(x1) ≠ f(x2)`，则称 `f` 为单射。

3. **双射**  
   若 `f` 既是单射又是满射，则称 `f` 为双射，也称一一映射。

### 17. 映射在不同数学分支中的名称

1. **一般名称**  
   映射也常称为算子。

2. **常见称法**  
   1. 若映射到数集，可称为泛函  
   2. 若 `X → X`，可称为变换  
   3. 若 `X → R`，可称为定义在 `X` 上的函数

### 18. 逆映射

1. **逆映射的定义**  
   若映射 `f: D → f(D)` 是单射，那么对每个 `y ∈ f(D)`，都有唯一的 `x ∈ D` 使得 `f(x) = y`。  
   因而可定义新的映射：  
   `g: f(D) → D`  
   这个映射称为 `f` 的逆映射，记作：  
   `f^(-1)`

2. **逆映射存在的前提**  
   映射必须是单射。

3. **易错点**  
   逆映射不是所有映射都有，必须先检查是否满足单射条件。

### 19. 复合映射

1. **复合映射的定义**  
   若有映射链：  
   `D --g--> D1 --f--> Y`  
   且满足：  
   `g(D) ⊂ D1`  
   则可定义复合映射：  
   `f ∘ g`

2. **计算公式**  
   `(f ∘ g)(x) = f(g(x))`

3. **复合映射成立的条件**  
   `g(D) ⊂ D1` 是构成复合映射不可缺少的条件。

### 20. 映射部分要点总结

1. **必须掌握的核心点**  
   1. 映射要求“每个元素都有唯一像”  
   2. 不同元素可以对应同一个像  
   3. 单射、满射、双射的定义要分清  
   4. 逆映射存在的前提是单射  
   5. 复合映射成立必须检查前一个映射的值域是否落入后一个映射的定义域中

2. **学习作用**  
   映射是函数概念的上一级抽象，函数本质上就是定义域为实数集子集的映射。

## 四、函数

### 21. 函数的定义

1. **函数**  
   设数集 `D ⊂ R`，则称映射  
   `f: D → R`  
   为定义在 `D` 上的函数，记作：  
   `y = f(x)，x ∈ D`

### 22. 函数的基本组成

1. **自变量**  
   `x` 称为自变量。

2. **因变量**  
   `y` 称为因变量。

3. **定义域**  
   `D` 称为函数的定义域。

4. **值域**  
   `f(D)` 称为函数的值域。

### 23. 函数图形

1. **图形定义**  
   函数图形是平面点集：  
   `C = {(x, y) | y = f(x)，x ∈ D}`

2. **本质理解**  
   函数图形本质上是满足函数关系的所有点组成的集合。

### 24. 函数定义域的含义

1. **定义域的本质**  
   定义域是使函数表达式有意义、并且实际问题也有意义的自变量集合。

2. **解题意义**  
   求函数定义域时，核心就是找出所有使表达式成立的 `x` 的取值范围。

### 25. 函数对应规律的表示方法

1. **解析法**  
   用公式表示函数关系。

2. **图象法**  
   用图形表示函数关系。

3. **列表法**  
   用表格列出对应关系。

### 26. 典型函数

1. **反正弦主值函数**  
   `y = arcsin x`  
   定义域：`[-1, 1]`  
   值域：`[-π/2, π/2]`

2. **绝对值函数**  
   `y = |x|`  
   定义域：`R`  
   值域：`[0, +∞)`

3. **符号函数**  
   `sgn x =`  
   `1,   x > 0`  
   `0,   x = 0`  
   `-1,  x < 0`

   定义域：`(-∞, +∞)`  
   值域：`{-1, 0, 1}`

4. **取整函数**  
   `y = [x]`  
   表示不超过 `x` 的最大整数。  
   例如：  
   `[2.5] = 2`  
   `[-3.4] = -4`

### 27. 函数部分要点总结

1. **必须理解的核心内容**  
   1. 函数是定义在实数集子集上的映射  
   2. 函数一定要有定义域、对应规则和值域  
   3. 值域是函数真正能够取到的值，不是随便写的范围  
   4. 函数图像是一个点集  
   5. 定义域必须从“表达式有意义”出发判断

2. **易错点**  
   1. 值域和陪域容易混淆  
   2. 分段函数求值时，先判断 `x` 落在哪一段  
   3. 不是任意关系都能叫函数，必须满足“每个 `x` 只有一个函数值”
